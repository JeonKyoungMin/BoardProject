# 전경민 프로젝트

## 프로젝트 개발환경
#### OS : Win10
#### IDE : Eclipse
#### DBMS : Mysql
#### DB처리 : JDBC
#### 미들웨어 : Tomcat v9.0
#### FrameWork : Spring@MVC v.5.3.2
#### 사용 라이브러리 : JSTL, hiberbate validation, dbcp2, MyBatis, Spring-security


## 기능
#### 1. 회원가입 / 회원탈퇴
#### 2. 로그인 / 로그아웃, 회원 정보찾기, 회원 정보수정
#### 3. 게시글 작성/ 수정/ 삭제/ 검색 , 댓글 작성
